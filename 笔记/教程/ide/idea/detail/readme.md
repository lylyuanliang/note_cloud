# IDEA插件详情

这里按插件分别记录功能、安装方式和使用场景，便于判断是否需要安装。

### 1. [Maven Helper](./1.maven%20helper.md)

> 用于分析 Maven 依赖冲突、查看依赖树和辅助处理依赖问题。

### 2. [MyBatisX](./2.MyBatisX.md)

> 用于 MyBatis Mapper、XML 和接口之间的跳转与代码生成辅助。

### 3. [Rainbow Brackets](./3.Rainbow%20Brackets.md)

> 用彩色括号提升嵌套代码的阅读体验。

### 4. [GitToolBox](./4.GitToolBox.md)

> 增强 Git 提交信息、分支状态和代码行责任人提示。

### 5. [SequenceDiagram](./5.SequenceDiagram.md)

> 根据代码生成调用时序图，辅助理解复杂调用链。

### 6. [Developer Tools](./6.Developer%20Tools.md)

> 提供编码、转换、格式化等开发辅助工具集合。

### 7. [Statistic](./7.Statistic.md)

> 统计项目代码行数、文件类型和代码分布。

### 8. [EasyApi](./8.EasyApi.md)

> 辅助生成和管理接口文档。

### 9. [Spring Debugger](./9.Spring%20Debugger.md)

> 辅助 Spring 项目调试和 Bean 相关信息查看。
