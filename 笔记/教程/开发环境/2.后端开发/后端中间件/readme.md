# 后端中间件

该目录用于整理后端项目常用中间件在麒麟 V10 上的安装、部署、配置和排障教程，覆盖手动部署与 Docker Compose 部署两类场景。

## 1.文档清单

### 1.1 [麒麟 V10 后端中间件手动安装教程](./1.麒麟V10后端中间件手动安装教程.md)

适用于在麒麟 V10 服务器上直接安装 JDK、MySQL、Redis、Kafka、Nacos、Nginx、Canal 等后端依赖。

### 1.2 [麒麟 V10 Docker 中间件部署教程](./2.麒麟V10-Docker中间件部署教程.md)

适用于在麒麟 V10 服务器上使用 Docker Compose 统一部署 MySQL、Redis、Kafka、Nacos、Nginx、Canal 等中间件。
