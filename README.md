# note_cloud

个人云笔记仓库，用于记录开发环境搭建、工具教程、后端技术学习、Docker compose 样例和日常排障经验。

## 云笔记自动保存方案

> - vscode
> - 安装插件 `Note Sync`
> - 导入 git 仓库
